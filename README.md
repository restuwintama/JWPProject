# ⛰️ Arunika Outdoor 

Sistem Informasi Penyewaan & Penjualan Alat Pendakian Berbasis Website.

---

## 🚀 Deskripsi
**Arunika Outdoor** adalah platform digital yang mengotomatisasi pengelolaan toko alat pendakian. Sistem ini menghubungkan Admin (pengelola Basecamp) dengan Pelanggan (pendaki) dalam satu ekosistem yang terintegrasi untuk proses sewa dan beli alat gunung.

## 🛠️ Tech Stack
*   **Backend:** PHP (Native)
*   **Database:** MySQL
*   **Frontend:** HTML5, CSS3 (Tailwind CSS)
*   **Assets:** FontAwesome / SVG Icons

## ⚙️ Fitur Utama

### 👤 Pelanggan (User)
- [x] **Registrasi & Login:** Validasi akun & *real-time alert*.
- [x] **Katalog Produk:** Fitur Sewa & Beli dengan *Modal Detail*.
- [x] **Keranjang Belanja:** Kelola jumlah barang & *check-out* aman.
- [x] **Manajemen Profil:** Edit profil & upload foto profil (opsional).
- [x] **Invoice:** Nota otomatis saat *checkout* berhasil.

### 👑 Administrator
- [x] **Dashboard:** Statistik pesanan harian & total member.
- [x] **Kelola Produk:** CRUD (Create, Read, Update, Delete) produk dengan **Upload Gambar**.
- [x] **Manajemen Pesanan:** Verifikasi (ACC/Tolak) pesanan yang masuk.
- [x] **Data Member:** Monitoring daftar pelanggan terdaftar.

---

## 💻 Cara Instalasi
1. Pastikan **XAMPP** terinstal dan modul **Apache** & **MySQL** berjalan.
2. Clone atau salin folder proyek ke `C:\xampp\htdocs\arunika`.
3. Buka browser dan akses `http://localhost/arunika`.
4. Database akan **dibuat secara otomatis** saat aplikasi diakses pertama kali.
